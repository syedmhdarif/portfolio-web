## Prompt handling

Before starting any non-trivial task, invoke the `prompt-architect` skill to classify effort (SIMPLE vs HIGH-EFFORT) and, for HIGH-EFFORT, build a structured 10-part prompt before executing. Skip for trivial mechanical asks (typo fixes, single-line edits, listing queries). When in doubt for design / copy / cross-cutting work, lean HIGH-EFFORT.
